# Digital Resume Website

A modern, responsive CV website built with Next.js and Tailwind CSS, inspired by the Ortiz Fluid template.

## Features

- Responsive design
- Smooth animations with Framer Motion
- Modern UI with Tailwind CSS
- SEO optimized
- Fast performance

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

1. Update the content in `app/page.tsx` with your personal information
2. Modify the colors in `tailwind.config.js` to match your preferred color scheme
3. Add your own sections or modify existing ones as needed

## Building for Production

```bash
npm run build
```

## Technologies Used

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- TypeScript
